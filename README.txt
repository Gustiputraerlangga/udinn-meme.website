Udin - Simple Landing Page
Files included:
- index.html
- styles.css
- script.js

How to use:
1. Download the files and open index.html in your browser to preview.
2. Customize text (supply, distribution, links) in index.html.
3. Replace placeholder links (Telegram, Twitter/X, Discord, buy link) with real URLs.
4. To deploy:
   - GitHub Pages: push to a repo and enable Pages from main branch.
   - Netlify / Vercel: drag-and-drop the folder or connect the repo.
5. Make sure to add the real contract address and DEX link once the token and contract are created.

Notes:
- This template is for informational and promotional purposes only. Not financial advice.
- Update tokenomics and legal disclaimers according to local regulations.
